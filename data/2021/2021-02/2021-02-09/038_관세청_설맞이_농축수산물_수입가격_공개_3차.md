# 설맞이 농축수산물 수입가격 공개(3차)

- 부처: 관세청
- 배포일: 02/09/2021 10:13:09
- NewsItemId: 156435963
- 원문: https://www.korea.kr/briefing/pressReleaseView.do?newsId=156435963&call_from=openData

## 본문

관세청은 설을 맞이하여 소비가 증가하는 주요 농축수산물* 66개 품목의 장바구니 물가 안정 지원을 위해 주 단위로 수입가격을 공개
 * 수입량이 연속성을 가지는 품목을 중심으로 명절 수요 등 고려 선정
 -> 설맞이 농축수산물 수입가격은 1차(1.26.), 2차(2.2.), 3차(2.9.) 총 3회에 걸쳐 공개
 -> 이번 3차 발표 자료는 전년도 설 연휴 1주~3주 전인 ’19년 12.27.∼’20년 1.16. 기간과 ‘21년 1.14.∼2.3. 기간을 비교 분석
1. 요 약
 분석기간 내 주요 농축수산물 33개 품목 상승, 33개 품목 하락
 -> (농산물) 36개 품목 중 20개 품목 상승, 16개 품목 하락

 -> (축산물) 10개 품목 중 5개 품목 상승, 5개 품목 하락

 -> (수산물) 20개 품목 중 8개 품목 상승, 12개 품목 하락

2. 주요 품목별 평균 수입가격* 현황
 * CIF가격에 관세 등 제세를 합친 금액을 중량으로 나눈 kg당 평균가격
-> (농산물)
 - 들깨(5,076원/kg, 102.7%), 양배추(신선, 냉장)(651원/kg, 86.0%), 양파(신선, 냉장)(1,033원/kg, 39.3%), 팥(건조)(2,666원/kg, 32.3%), 당근(신선, 냉장)(599원/kg, 23.3%) 등은 상승
 - 아몬드(탈각)(5,492원/kg, △33.7%), 호두(탈각)(6,112원/kg, △33.5%), 무(신선, 냉장)(267원/kg, △33.1%), 참깨(2,405원/kg, △27.6%), 밤(신선, 건조-무탈각)(2,831원/kg, △17.4%) 등은 하락
-> (축산물)
 - 닭날개(냉동)(3,731원/kg, 23.5%), 기타돼지고기(냉동)(3,278원/kg, 9.1%), 삼겹살(냉동)(5,920원/kg, 3.9%) 등은 상승
 - 닭다리(냉동)(2,105원/kg, △25.5%), 소갈비(냉동)(7,702원/kg, △23.6%), 뼈없는소고기(냉동)(7,177원/kg, △8.2%) 등은 하락
-> (수산물)

 - 임연수어(냉동)(2,495원/kg, △27.8%), 아귀(냉동)(1,526원/kg, △25.4%), 오징어(냉동)(3,390원/kg, △21.1%), 고등어(냉동)(2,347원/kg, △16.5%), 조기(냉동)(4,672원/kg, △14.8%) 등은 하락
