# gov-press-md

정부 보도자료를 API로 수집해 Markdown으로 정리하는 프로젝트.

## 목표
- 정책브리핑 보도자료 API(XML + HTML 혼합 응답)를 사람이 읽기 좋고 에이전트가 처리하기 좋은 Markdown으로 변환
- 날짜별/부처별 아카이브 구축
- 공공형 검색, 브리핑, 비교, 요약 작업의 기반 데이터셋 제공

## 현재 범위
- 정책브리핑 보도자료 API 기반 수집
- 1건 1파일 Markdown 생성
- 날짜별 폴더 및 일자별 README 인덱스 생성

## 디렉터리 구조
```text
gov-press-md/
  README.md
  data/
    2026/
      2026-04/
        2026-04-01/
        2026-04-02/
        ...
  scripts/
  docs/
```

## Markdown 파일 형식
각 보도자료 파일에는 다음 정보가 들어간다.
- 제목
- 부처
- 배포일
- NewsItemId
- 원문 링크
- 본문 정리본
- 출처
